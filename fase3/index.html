Etiquetas

Definicion de Etiquetas HTML

HTML Es un código que se compone por diferentes etiquetas las cuales el navegador interpreta y permite crear o modificar la apariencia de un documento de hipertexto.

Las etiquetas y los atributos conforman el esqueleto de marcas del lenguaje HTML.

Las etiquetas inician con el simbolo menor que ''<'' y terminan con el mayor que ''>''

Atributos

Definicion de Atributos HTML

Una etiqueta puede tener uno o varios atributos, estos se diferencian porque estan dentro de las etiquetas.

Los Atributos especifican la accion que se desea realizar, en cambio las etiquetas el elemento que vamos a describir y mostrar.
